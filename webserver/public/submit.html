Your config file has been submitted!

Now that you have created your "config" for the smart mirror you will need to upload some photos of yourself so that the mirror can recognise you on use. Please upload photos using the upload tool below and make sure that the photo only has your face in it. If you wear glasses I would suggest uploading one with glasses and one without to make facial recognition easier. When you upload the photos you will navigated to a confirmation page. The next page will contain a finish button which will take the data from the photos and delete the photos from the system. The facial data is then only stored in a singular file called dataset_files.dat

Make sure to select all photos when you click choose files below.

If you have upload photos before and you just wanted to change your config settings then you can exit this page.
